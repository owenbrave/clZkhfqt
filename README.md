## 前言

您好！欢迎查看“高校体育场管理系统+SSM”项目，这是一个基于Java语言和SSM框架的体育场管理系统。该项目旨在帮助高校更高效地管理体育场地资源，提供便捷的服务。以下是该项目的详细介绍。

## 内容介绍

本项目主要功能包括：场地预约、场地查询、场地使用情况统计、用户管理等。通过使用Spring、SpringMVC和MyBatis框架，实现了前后端分离，提高了开发效率和项目可维护性。前端采用Uniapp框架，实现了一套代码多端适配，方便用户在微信小程序、Web端等使用。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
